# 🍔 BitByte - Food Delivery App

BitByte is a modern **Food Delivery Android application** built using **Jetpack Compose** and **Clean Architecture**, designed to simulate real-world production-level apps with scalable architecture and smooth user experience.

---

## 🚀 About This Project

This project is built to demonstrate my understanding of:

- Scalable Android architecture  
- Modern UI development using Jetpack Compose  
- Managing complex data flow between API, local storage, and cloud  
- Real-world app features like authentication, cart, and payments  

---

## ✨ Features

- 🌱 Veg / Non-Veg food filter  
- 🛒 Cart management with Room Database  
- 💳 Razorpay payment integration (Test Mode)  
- 🔐 Firebase Authentication  
- ☁️ Firestore for storing order history  
- ⚙️ DataStore for user preferences (address, UID, Veg mode)  
- 📡 API integration using Retrofit (JSONBin)  
- 🎨 Smooth UI with Jetpack Compose & Lottie animations  
- 📦 Offline support using local caching  

---

## 🏗️ Architecture

- Clean Architecture + MVVM  
- Separation of Data, Domain, and Presentation layers  
- Scalable and maintainable code structure  
- Dependency Injection using Hilt  

---

## 🛠️ Tech Stack

- **Language:** Kotlin  
- **UI:** Jetpack Compose  
- **Architecture:** MVVM + Clean Architecture  
- **DI:** Hilt  
- **Networking:** Retrofit + Gson (JSONBin API)  
- **Local Storage:** Room Database  
- **Preferences:** DataStore  
- **Backend:** Firebase Auth & Firestore  
- **Payments:** Razorpay  
- **Image Loading:** Coil  
- **Animations:** Lottie  
- **Navigation:** Navigation Component  

---

## 📱 Data Flow

- API Data → Fetched using **Retrofit (JSONBin)** → Displayed in UI  
- Cart Data → Stored locally using **Room Database**  
- User Preferences → Stored using **DataStore**  
- Order History → Stored in **Firestore after successful payment**
